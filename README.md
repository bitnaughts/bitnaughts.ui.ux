# BitNaughts' UI/UX Module

All things related to the in-game UI, UX, and UI-related animations are controlled via this module.

... more to come!

BitNaughts is a true open-source project. If you'd like to contribute to the interpreter, feel free to make a pull request! 

## Concepts

Monitor panels enable viewing of datasets. Consoles can connect to Monitors to transmit data to be visualized. A common use-case would be a Console executing code to calculate nearest enemies, then visualizing a point-cloud of results on a connected Monitor

## Naming Convention

  - Classes: PascalCase
  - Static Methods: PascalCase
  - Non-static Methods: camelCase
  - Fields: snake_case
  - Constants: ALL_UPPER_CASE
